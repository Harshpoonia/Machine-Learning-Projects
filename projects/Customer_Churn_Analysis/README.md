# Customer Churn Prediction

## Project Overview

Customer churn prediction helps businesses identify customers who are likely to leave a service.

This project uses machine learning techniques to predict whether a customer will churn based on demographic and service-related features.

---

## Objective

Build a classification model that predicts customer churn.
